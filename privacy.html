3 · Server Log Files

GitHub, as hosting provider, automatically collects and stores information in server log files that your browser transmits automatically. This may include:

  • IP address of the requesting device
  • Date and time of access
  • Name and URL of the accessed file
  • Referring website (referrer URL)
  • Browser type and operating system
  • Name of your Internet service provider

This data is not merged with other data sources and is used solely for technical operation and security purposes by GitHub as the hosting provider.

Legal basis: Art. 6(1)(f) GDPR – legitimate interest in detecting errors and ensuring secure operation

5 · Browser Extensions – Data Handling

✓ No server-side data collection

All browser extensions published under rheysoft process data exclusively locally within your browser. No user data is transmitted to or stored on servers operated by rheysoft.

Locally stored data

Some extensions use the browser's built-in storage.local API to save user preferences (e.g., configuration settings) locally in your browser. This data:

  • remains exclusively on your device;
  • is never transmitted to any external server;
  • can be deleted at any time by uninstalling the extension or via your browser settings.

Permissions used

Each extension only requests the minimum permissions necessary for its functionality. The specific permissions and their purpose are documented in the respective store listing.

External services (if applicable)

If an extension connects to any external API or third-party service (e.g., for market data), this will be explicitly disclosed in the product description in the browser store and in the product-specific privacy policy section.

Legal basis: No GDPR-relevant processing occurs for locally-stored data (no personal data transmitted to the controller).
Store purchases: When downloading or purchasing via browser stores, the respective store operator's privacy policy applies (Google, Mozilla, Microsoft).

Data collected by store platforms

When you install an extension from a browser store, the store platform (Google, Mozilla, Microsoft) may collect and process data such as installation events and crash reports in accordance with their respective privacy policies.
