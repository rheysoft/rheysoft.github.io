## 5. Browser Extensions – Data Handling

### ✅ No Data Collected or Transmitted

All browser extensions published under **rheysoft** on the Chrome Web Store, Firefox Add-ons (AMO), and Microsoft Edge Add-ons:

- **Process all data exclusively locally within your browser**
- **Do not transmit any personal data to rheysoft servers**
- **Do not collect usage statistics or analytics**
- **Do not use any tracking mechanisms**

### Local Storage

Some extensions use the browser's `storage.local` API to save your preferences (e.g., settings, configuration) locally. This data:

- remains exclusively on your device
- is never transmitted externally
- can be deleted at any time by uninstalling the extension or clearing browser storage

### Permissions

Each extension requests only the minimum permissions necessary for its functionality. Permissions and their purpose are documented in the respective store listing.

### External Services (if applicable)

If a specific extension connects to an external API (e.g., for market data), this is explicitly disclosed in the product's individual store description and in a product-specific privacy notice.

### Store Platform Data

When you install an extension from a browser store, the store operator (Google, Mozilla, Microsoft) may collect data such as installation events. This is governed by their respective privacy policies.
